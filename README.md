# 「记」

摒除杂质，专注创作

- 点击进入编辑，底部工具栏可以方便地进行缩进、撤销和粘贴操作
- 向下滑动进入历史笔记，向上滑动快速创建新笔记

无广告、无联网权限，本地备份和恢复数据。
